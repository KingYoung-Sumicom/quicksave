How it works

Three pieces, one encrypted channel. The relay routes frames; only the PWA and your laptop hold keys.

Browser
PWA

React app on your phone. Holds half the key.

Stateless
Relay

Routes encrypted frames. Cannot decrypt. Self-hostable.

Daemon
Agent

Runs on your laptop. Owns NaCl keys, runs git + Claude.

End-to-end encrypted

NaCl keypair handshake between PWA and agent. The relay is a stateless WebSocket router — it never sees plaintext.

Git workflows on mobile

Browse status, hunk-by-hunk diffs, stage, commit, push — with the same keystrokes you'd use at your desk.

Drive Claude Code remotely

Spawn Claude Code CLI sessions, review tool calls inline, approve permissions, and resume from anywhere.

Quick start

Two steps: install the agent on the machine you want to control, then pair from your phone.

  1. 1

    Install the agent on your dev machine 

    # requires Node.js 20+
npm install -g @sumicom/quicksave
cd /path/to/your/repo
quicksave

    Prints a pairing URL and QR code. Keeps a background daemon running.

  2. 2

    Pair from your phone

    Open quicksave.dev and scan the QR code. Add to home screen for the full PWA experience. Everything from this point is end-to-end encrypted.

Self-host the relay

Don't want to depend on the public relay? Build the Docker image and point your PWA at it. 

docker build -f apps/relay/Dockerfile -t quicksave-relay .
docker run -p 8080:8080 quicksave-relay

Full deployment checklist in docs/relay/deployment.md.
